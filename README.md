This is a simple C# demo for [stable-diffusion.cpp](https://github.com/leejet/stable-diffusion.cpp)

In this demo, stable-diffusion.dll is for cuda12, and you can replace it for your PC environment. 
The dll can be downloaded from [stabel-diffusion.cpp/release](https://github.com/leejet/stable-diffusion.cpp/releases)

Text to Image model and Image to Image demo model can download from [Stable Diffusion v1.5](https://huggingface.co/runwayml/stable-diffusion-v1-5).

Upscale model can download from [RealESRGAN](https://github.com/xinntao/Real-ESRGAN/releases/download/v0.2.2.4/RealESRGAN_x4plus_anime_6B.pth).
